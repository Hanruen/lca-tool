# 工程材料碳足跡計算工具 — 使用說明

## 第一次使用（只需做一次）

1. 解壓縮收到的 zip 檔案
2. 對「啟動碳足跡計算工具.command」按右鍵 → 選「開啟」
   （第一次需要右鍵開啟，之後就可以直接雙擊）
3. 跳出安全性警告時，點「開啟」確認
4. 等待套件安裝完成（約 1-2 分鐘），瀏覽器會自動開啟

## 之後每次使用

直接雙擊「啟動碳足跡計算工具.command」，瀏覽器會自動開啟。

## 注意事項

- 使用期間請勿關閉自動出現的終端機視窗
- 關閉工具：直接關閉終端機視窗即可
- 若瀏覽器沒有自動開啟，請手動輸入：http://localhost:8501

## 系統需求

- macOS
- Python 3（若未安裝請至 https://www.python.org 下載）
- 網路連線（僅第一次安裝套件時需要）
