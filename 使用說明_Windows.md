# 工程材料碳足跡計算工具 — Windows 使用說明

## 第一次使用（只需做一次）

1. 解壓縮收到的 zip 檔案
2. 雙擊「啟動碳足跡計算工具.bat」
3. 等待套件安裝完成（約 1-2 分鐘）
4. 瀏覽器會自動開啟工具

## 之後每次使用

直接雙擊「啟動碳足跡計算工具.bat」，瀏覽器自動開啟。

## 注意事項

- 使用期間請勿關閉自動出現的黑色命令列視窗
- 關閉工具：直接關閉黑色視窗即可
- 若瀏覽器沒有自動開啟，請手動輸入：http://localhost:8501

## 系統需求

- Windows 10 / 11
- Python 3（若未安裝請至 https://www.python.org 下載，安裝時勾選「Add Python to PATH」）
- 網路連線（僅第一次安裝套件時需要）
